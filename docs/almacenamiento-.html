Sobre el almacenamiento abierto de datos

7.3 Almacenamiento.

  • Subir los datos originales, los datos etiquetados y la documentación Open Science Framework

    • Indicar que el contenido del proyecto son datos.
    • Agregar una descripción suficiente.
    • Indicar palabras clave. Incluir el termino “data”.
    • Para mayor información sobre este proceso visite el siguiente tutorial

  • Resguardar los datos en algun servidor local de confianza.

    • Evaluar la posibilidad de almacenar en servidores de univercidades y/o bibliotecas. Es importante resguardar los datos despues de su curatoria.
