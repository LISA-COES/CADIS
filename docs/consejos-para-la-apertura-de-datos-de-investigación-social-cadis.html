Sobre el almacenamiento abierto de datos

Sección 5 Consejos para la Apertura de Datos de Investigación Social (CADIS)

En esta sección se presenta una guía sobre cómo y dónde subir los datos producidos por las investigaciones de ciencias sociales, siguiendo los estándares internacionales revisados (Secciones 3-4) y aprovechando los aprendizajes de las buenas experiencias de almacenamiento (Sección 5).

Para publicar los datos de investigación proponemos tres momentos, los cuales serán abordados cada uno en las sub-secciones siguientes:

  • Documentar

¿Qué información y documentos debo subir junto a los datos?

Esta sub-sección señala los documentos necesarios que deben ser adjuntados. Por ello implica recolectar, planificar y/o producir toda la información necesaria para comprender el contenido y las características de los datos almacenados. El investigador frente a esta sección debería generar una carpeta digital que contenga los datos y la documentación requerida ordenada de un modo estándar.

  • Adecuar

¿Qué debo hacer con mis datos antes de abrirlos?

Se indican las labores para hacer los datos utilizables a todos, y por ello implica transformar los documentos a ciertos formatos, procurar la confidencialidad y preparar bases de datos para ser usadas por distintos softwares. El investigador debería finalizar con datos preparados para ser publicados.

  • Publicar

¿Dónde y cómo subo mis datos para abrirlos?

Esta sub-sección indica donde subir los datos y expone un ejemplo de cómo hacerlo. También se discuten las ventajas y desventajas de distintas plataformas para publicar los datos. El investigador.
