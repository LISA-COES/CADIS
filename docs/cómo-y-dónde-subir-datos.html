Sobre el almacenamiento abierto de datos

Módulo 6 Cómo y dónde subir datos

La mayoría de las buenas experiencias presentadas sobre el almacenamiento de datos requieren para poder subir los datos de la investigación el estar afiliado a una institución asociada o el pagar por dicho servicio. Un caso contrario es el de los repositorios MOVID y el laboratorio de acceso a la justicia, pues estos almacenamientos de datos y productos de investigación utilizaron plataformas de ciencia abierta, como lo son Git Hub y OSF.

Entre las distintas páginas web que prestan servicios similares se puede decir que OSF es una de las más completas, en tanto permite hacer identificadores doi para cumplir con los criterios FAIR, permite generar metadatos de forma sencilla, puede adaptarse a distintas necesidades individuales o institucionales, además de poseer una diversidad de herramientas para registrar las distintas etapas del proceso de investigación. Otra ventaja considerable de OSF es que como se sugiere en libro de Corti (2019), permite sincronizar distintos repositorios, lo cual junto con dejar versiones en servidores locales, puede ayudar a preservar los datos y productos de la investigación incluso después de que la plataforma esté operando.

References

Corti, Louise. 2019. Managing and Sharing Research Data: A Guide to Good Practice. 2nd edition. Thousand Oaks, CA: SAGE Publications.
